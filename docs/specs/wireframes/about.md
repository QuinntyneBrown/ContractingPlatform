## Component Specifications

### Story Section
- **Layout:** 2-column on desktop (40% image, 60% text)
- **Image:** Rounded corners, subtle shadow
- **Text:** Left-aligned, comfortable line-height

### Values Grid
- **Desktop:** 5 columns equally spaced
- **Mobile:** 2x2 grid + 1 centered
- **Icon Size:** 48px
- **Card Padding:** 24px

### Team Grid
- **Desktop:** 3 columns
- **Mobile:** 1 column, stacked
- **Photo:** 150x150px, circular or rounded
- **Card:** Centered text, subtle shadow

### Credentials Row
- **Layout:** Horizontal scroll on mobile
- **Logo Height:** 60px maximum
- **Grayscale:** Default, color on hover

### Service Areas Map
- **Width:** 100% of container
- **Max-width:** 800px, centered
- **Borough Labels:** Positioned on map
- **Neighborhoods:** Listed below in columns
