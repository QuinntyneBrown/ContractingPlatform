## Component Specifications

### Emergency Banner
- **Background:** #FEF3C7 (amber-100) or #FEE2E2 (red-100)
- **Border:** 4px left solid #F59E0B or #EF4444
- **Icon:** Warning/alert icon
- **Phone:** Large, bold, clickable

### Contact Form
- **Container:** White background, rounded corners, shadow
- **Padding:** 40px (desktop), 24px (mobile)
- **Field Height:** 48px for inputs
- **Field Font Size:** 16px (prevents zoom on iOS)

### Form Fields
- **Labels:** 14px, font-weight 500, color gray-700
- **Inputs:** Full width, 1px border gray-300
- **Focus:** 2px border primary color
- **Error:** 2px border red-500, error message below

### Contact Info Cards
- **Layout:** Vertical stack with icon left
- **Icon:** 32px, primary color background circle
- **Text:** Link styling for clickable items
- **Hover:** Slight background highlight

### Business Hours
- **Layout:** Simple list
- **Current Status:** Green/red indicator if currently open/closed
- **Emergency:** Highlighted with icon

### Map
- **Height:** 400px (desktop), 300px (mobile)
- **Border Radius:** 12px
- **Controls:** Standard Google Maps
- **Marker:** Custom pin with company logo (optional)

### Submit Button
- **Width:** 100%
- **Height:** 56px
- **Background:** Primary color
- **Hover:** Darker shade
- **Loading:** Spinner, disabled state

### File Upload
- **Style:** Dashed border, drag & drop zone
- **Hover:** Border color change
- **Preview:** Thumbnails with remove button
- **Progress:** Upload progress bar

## Accessibility Notes

- All form fields have visible labels
- Error messages linked to fields via aria-describedby
- Focus indicators visible
- Phone number linked with tel: protocol
- Email linked with mailto: protocol
- Map has text alternative for address
