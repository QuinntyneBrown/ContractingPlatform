## Component Specifications

### Filter Bar
- **Position:** Sticky below header (optional)
- **Background:** White with subtle shadow
- **Pills:** Border-radius 9999px (full)
- **Active:** Primary background, white text
- **Inactive:** Gray border, gray text

### Project Cards
- **Aspect Ratio:** 4:3 for image
- **Border Radius:** 12px
- **Shadow:** 0 4px 6px rgba(0,0,0,0.1)
- **Hover:** Transform scale(1.02), shadow increase
- **Category Badge:** Small pill in card content

### Lightbox
- **Background:** rgba(0,0,0,0.95)
- **Max Image Width:** 80vw
- **Max Image Height:** 70vh
- **Thumbnails:** 60x60px, border on active
- **Navigation:** 48px arrows, white

### Before/After Slider
- **Handle:** 48px circle with arrows
- **Divider:** 4px white line
- **Labels:** Semi-transparent overlays

### Load More Button
- **Style:** Outlined primary
- **Width:** Auto, centered
- **Hover:** Filled primary

## Interactions

### Card Click
- Opens lightbox with selected project
- Scrolls to project in lightbox

### Filter Click
- Immediate filter with smooth transition
- Count updates in real-time
- Animation: Fade out/in or scale

### Lightbox Navigation
- Arrow keys for navigation
- Swipe on mobile
- ESC to close
- Click outside to close

### Lazy Loading
- Load 12 projects initially
- Load 12 more on "Load More" click
- Or infinite scroll (optional)
